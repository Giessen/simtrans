This folder contains ill-defined model files to check validation ability of simtrans-checker

valid.wrl: valid model
invalid-same-jointid.wrl: invalid model with same joint ID
invalid-same-jointname.wrl: invalid model with same joint name
invalid-same-limit.wrl: invalid model with same joint upper and lower limit.
invalid-opposite-vlimit.wrl: invalid model with opposite joint velocity limit.
